# Elfsight Test Task (Rick and Morty API)

Development task of the Elfsight technical assessment. It focuses on building a functional character browser using the Rick and Morty API.

## Technology Stack

- React 18
- Styled Components 5
- Axios 0.27

## Requirements
- Node.js >= 16
- npm >= 8

## Start

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run start
```

Open [http://localhost:3000](http://localhost:3000) in browser.
